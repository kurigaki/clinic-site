# clinic-site

旭川市の内科・糖尿病内科クリニック向け、静的HTMLサイトのテンプレートです。

## プロジェクト概要

開院予定または既存のクリニックが、すぐに本番運用できることを目的として制作しました。
フレームワーク不使用のバニラHTML/CSSで構成されており、サーバー環境を選ばず軽量に動作します。

## 目的

- 糖尿病・生活習慣病・一般内科を専門とする女性医師のクリニックを想定したWeb集患サイト
- 「怒られそう」「大げさかな」といった来院障壁を取り除き、予約・電話への導線を最大化する
- モバイルファーストのレスポンシブデザインで、30〜70代の幅広い患者層に対応
- SEO（構造化データ・canonical・OGP）とアクセシビリティ（WCAG準拠）を両立

## 技術スタック

- HTML5 / CSS3（カスタムプロパティ使用）
- バニラJavaScript（ナビゲーション開閉のみ）
- JSON-LD 構造化データ（MedicalClinic / Physician / MedicalCondition）
- フレームワーク・ライブラリ不使用

## ページ構成

| ファイル | ページ名 |
|---|---|
| `index.html` | トップページ |
| `about.html` | 医院紹介 |
| `doctor.html` | 医師紹介 |
| `first-visit.html` | 初診の方へ |
| `faq.html` | よくある質問 |
| `access.html` | アクセス・診療時間 |
| `treatment/index.html` | 診療内容一覧 |
| `treatment/diabetes.html` | 糖尿病内科 |
| `treatment/lifestyle.html` | 生活習慣病 |
| `treatment/general.html` | 一般内科・発熱外来 |

## カスタマイズ

各HTMLファイル内の `【クリニック名】` `【電話番号】` `【予約URL】` `【ドメイン名】` 等のプレースホルダーを、実際のクリニック情報に置き換えてください。
各ページ末尾のHTMLコメントに、差し替えが必要な箇所の一覧を記載しています。
